### diff
* code
```
def f(x):
    # return x*x
    return x**3

dx = 0.001 /求逼近

def diff(f, x):
    df = f(x+dx)-f(x)
    return df/dx

print('diff(f,2)=', diff(f, 2))
```

### e (自然常數、自然底數)
* e的微分是自己，積分也是自己
* 執行結果會趨近於e
* code
```
def e(n):
	return (1+1.0/n)**n

for i in range(100):
	n = (i+1)*100.0
	print('n=', n, 'e(n)=', e(n))
```
